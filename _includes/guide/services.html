Services

The library contains some services which could be helpful additionally.


Serialization Service

The TraktSerializationService provides methods for serializing and deserializing TraktApiSharp objects.
Objects will be serialized as a JSON string.
Note: This service supports currently only TraktAuthorization.


Language Service

The TraktLanguageService provides methods for looking up the language and country names for language codes, that are returned by the Trakt API.
