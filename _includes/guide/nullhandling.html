Null Handling

The Trakt API might not return all data, especially not, when you are using extended info.
Therefore, you should check object properties, if they are null or not. {% highlight csharp %} var user = client.Users.GetUserProfileAsync("username"); var userAvatarUrl = user.Images?.Avatar?.Full; // will be null // with images var user = client.Users.GetUserProfileAsync("username", new TraktExtendedInfo { Images = true }); var userAvatarUrl = user.Images?.Avatar?.Full; // could be null {% endhighlight %}
