Namespaces

This is an overview of all namespaces in the TraktApiSharp library.

Namespace Content
TraktApiSharp Global namespace containing the TraktClient.
TraktApiSharp.Authentication Contains the authentication modules for OAuth- and Device-Authentication and classes for holding authorization information.
TraktApiSharp.Enums Contains all enumerations.
TraktApiSharp.Exceptions Contains all exceptions.
TraktApiSharp.Extensions Contains extension methods for DateTime and string.
TraktApiSharp.Modules Contains all library modules for accessing the Trakt API endpoints.
TraktApiSharp.Objects.Basic Contains all classes, which represent JSON objects, that are mainly used by other JSON objects.
TraktApiSharp.Objects.Get Contains all classes, which represent JSON objects, that are returned by the Trakt API.
TraktApiSharp.Objects.Post Contains all classes, which represent JSON objects, that can be sent to the Trakt API.
TraktApiSharp.Services Contains Serialization- and Language-Service.
TraktApiSharp.Utils Contains the utility classes Pair and Range.
