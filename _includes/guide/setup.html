Basic Setup

The main entry point to the library is an TraktClient instance. There are three different ways to create a new instance of it and which way to choose depends on how you want to use the library.

If you just want to use requests, which do not require OAuth authorization of Trakt.tv, you will only have to provide the client id of your Trakt application: {% highlight csharp %} var client = new TraktClient("Your Trakt Client Id"); // or var client = new TraktClient { ClientId = "Your Trakt Client Id" }; {% endhighlight %} If you also want to authenticate your application users, the client secret of your Trakt application will also be needed additionally: {% highlight csharp %} var client = new TraktClient("Your Trakt Client Id", "Your Trakt Client Secret"); // or var client = new TraktClient { ClientId = "Your Trakt Client Id", ClientSecret = "Your Trakt Client Secret" }; {% endhighlight %} If you want to use requests, which do require OAuth authorization, you will also have to provide an OAuth access token. Read the Authentication section on how to get a new Trakt.tv OAuth authorization, including access token and refresh token, or provide an existing access token: {% highlight csharp %} var client = new TraktClient("Your Trakt Client Id", "Your Trakt Client Secret") { Authorization = TraktAuthorization.CreateWith("An existing access token") }; // or var client = new TraktClient { ClientId = "Your Trakt Client Id", ClientSecret = "Your Trakt Client Secret", Authorization = TraktAuthorization.CreateWith("An existing access token") }; {% endhighlight %} Note: A client id for your Trakt application will always be needed.
