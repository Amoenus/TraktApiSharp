Installation

The recommended way to install TraktApiSharp is to use NuGet, but you can also directly link to a TraktApiSharp.dll file or build the library from source code.


NuGet

Navigate into your projectâ€™s directory and install TraktApiSharp with NuGet command line: Install-Package TraktApiSharp
Or open your project / solution in Visual Studio and use the Visual Studio Package Management and search for trakt.


DLL Binaries

If you want to link a TraktApiSharp.dll directly, you'll find the DLL in the Bin directory in the ZIP Archive of any specific version on the Downloads page.
Note: The DLL file only works with .NET (C# / Visual Basic) projects.


Build from Source

The library can only be built inside of Visual Studio. You need at least Visual Studio 2015 with C# 6 support. The ZIP Archive of any specific version contains the source code of the library or you clone the Github repository.

To actually build the library, open the TraktApiSharp.sln in Visual Studio and build the project TraktApiSharp in the directory Source\Lib. After the build was successful, you can find the library DLL in the directory Build\Lib\[Debug|Release].
