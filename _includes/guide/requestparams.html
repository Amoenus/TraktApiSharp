Request Parameters

Many methods in the modules accept extended info specifications, filters and / or collections with multiple object ids.


Extended Info

An TraktExtendedInfo instance can be created to specify, how much data should be retrieved for a request.
It is possible to create just one instance of it and use it for each request or to create a new instance of it for every new request.

To create an instance of TraktExtendedInfo, there are basically two ways. {% highlight csharp %} var extendedInfo = new TraktExtendedInfo { Full = true, Images = true }; // or var extendedInfo = new TraktExtendedInfo().SetFull().SetImages(); {% endhighlight %}

TraktExtendedInfo has a fluent interface which allows you to chain its method calls like in the following example. {% highlight csharp %} var extendedInfo = new TraktExtendedInfo().SetFull().SetImages().SetEpisodes().ResetFull().SetNoSeasons().Reset(); {% endhighlight %}
Filters

There are four different filters you can use.

  • TraktCalendarFilter for all methods in the TraktCalendarsModule
  • TraktMovieFilter for some methods in the TraktMoviesModule
  • TraktShowFilter for some methods in the TraktShowsModule
  • TraktSearchModule for the GetTextQueryResultsAsync() method in the TraktSearchModule
Each filter has a fluent interface which allows you to chain its method calls.
For example, create an TraktCalendarFilter instance like this. {% highlight csharp %} var filter = new TraktCalendarFilter().WithQuery("calendar movie") .WithYears(2016) .WithGenres("drama", "fantasy") .WithLanguages("en", "de") .WithCountries("us") .WithRuntimes(30, 60) .WithRatings(80, 95); {% endhighlight %} And edit it like this. {% highlight csharp %} filter.AddGenres("action", "science-fiction") .AddLanguages("fr"); {% endhighlight %} All methods, which begin with With... will overwrite existing values. If you want to add values, you must use a method, which begins with Add....
With Clear() you can remove all current values in a filter.


Multiple Ids

The methods TraktMoviesModule.GetMultipleMoviesAsync(), TraktShowsModule.GetMultipleShowsAsync(), TraktSeasonsModule.GetMultipleSeasonsAsync(), TraktEpisodesModule.GetMultipleEpisodesAsync(), TraktPeopleModule.GetMultiplePersonsAsync() and TraktUsersModule.GetMultipleCustomListsAsync() require each a collection of mutliple ids of objects, which you want to retrieve, as an argument.

Method Collection
TraktMoviesModule.GetMultipleMoviesAsync() TraktMultipleObjectsQueryParams
TraktShowsModule.GetMultipleShowsAsync() TraktMultipleObjectsQueryParams
TraktSeasonsModule.GetMultipleSeasonsAsync() TraktMultipleSeasonsQueryParams
TraktEpisodesModule.GetMultipleEpisodesAsync() TraktMultipleEpisodesQueryParams
TraktPeopleModule.GetMultiplePersonsAsync() TraktMultipleObjectsQueryParams
TraktUsersModule.GetMultipleCustomListsAsync() TraktMultipleUserListsQueryParams
TraktMultipleObjectsQueryParams
{% highlight csharp %} var queryParams = new TraktMultipleObjectsQueryParams { // { id[, extended info] } "id-1", { "id-2", new TraktExtendedInfo { Full = true } }, "id-3" }; // or var queryParams = new TraktMultipleObjectsQueryParams(); queryParams.Add("id-1"); queryParams.Add("id-2", new TraktExtendedInfo { Full = true }); queryParams.Add("id-3"); {% endhighlight %}
TraktMultipleSeasonsQueryParams
{% highlight csharp %} var queryParams = new TraktMultipleSeasonsQueryParams { // { show-id, seasonnumber[, extended info] } { "show-id-1", 1 }, { "show-id-2", 3, new TraktExtendedInfo { Full = true } }, { "show-id-3", 2 } }; // or var queryParams = new TraktMultipleSeasonsQueryParams(); queryParams.Add("show-id-1", 1); queryParams.Add("show-id-2", 3, new TraktExtendedInfo { Full = true }); queryParams.Add("show-id-3", 2); {% endhighlight %}
TraktMultipleEpisodesQueryParams
{% highlight csharp %} var queryParams = new TraktMultipleEpisodesQueryParams { // { show-id, seasonnumber, episodenumber[, extended info] } { "show-id-1", 1, 1 }, { "show-id-2", 3, 5, new TraktExtendedInfo { Full = true } }, { "show-id-3", 2, 1 } }; // or var queryParams = new TraktMultipleEpisodesQueryParams(); queryParams.Add("show-id-1", 1, 1); queryParams.Add("show-id-2", 3, 5, new TraktExtendedInfo { Full = true }); queryParams.Add("show-id-3", 2, 1); {% endhighlight %}
TraktMultipleUserListsQueryParams
{% highlight csharp %} var queryParams = new TraktMultipleUserListsQueryParams { // { username, list-id } { "username-1", "list-id-1" }, { "username-2", "list-id-3" }, { "username-3", "list-id-5" } }; // or var queryParams = new TraktMultipleUserListsQueryParams(); queryParams.Add("username-1", "list-id-1"); queryParams.Add("username-2", "list-id-3"); queryParams.Add("username-3", "list-id-5"); {% endhighlight %}
